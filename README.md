# express-auth
A simple express app with jwt authentication

To install
1. Run <code>yarn install</code>
2. Duplicate the .env.example file and add your own config
3. Play with it ! 

<h2>Routes</h2>
<ul>
  <li>/api/user</li>
  <li>/api/auth/singup</li>
  <li>/api/auth/signin</li>
  <li>/api/auth/admin</li>
</ul>
